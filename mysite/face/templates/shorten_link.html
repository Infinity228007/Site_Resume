Description:

This project is my virtual portfolio and personal resume page. It is created using modern web technologies and designed to provide an overview of my experience, skills, and projects.

Features:

  • Profile: Detailed information about me, my education, and experience.
  • Resume: My up-to-date resume in a convenient format.
  • Projects: A list of projects I've worked on, with descriptions, images, and links to GitHub repositories.
  • Navigation: Page navigation using a menu in the site's header.

Technologies:

  • Django: The foundation of the web application.
  • HTML/CSS: Page layout and styling.
  • JavaScript: Additional interactive elements.
  • GitHub: Code storage and project management.

Goal:

The goal of this project is to provide a convenient way to showcase my experience and skills to future employers and colleagues, as well as to demonstrate my web development skills.
